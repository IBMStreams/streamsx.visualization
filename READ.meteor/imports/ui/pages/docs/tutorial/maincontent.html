Create your READ App

In this tutorial, we will create and share a READ app with a responsive real-time dashboard. The app will use visualizations bundled out of the box with READ as well as new kinds of visualizations which we will design in the playground. The app will pull data from real-time REST APIs.

Follow these steps to create your READ app. We recommend opening this tutorial in a separate browser tab while working on your READ app.

Create App

Click on the App link . Provide an app name (we recommend Movies). Click create .
