Rapidly build and share real-time dashboards. Learn advanced concepts and techniques.

Create your READ App

In this tutorial, we will create and share a READ app with a responsive real-time dashboard. The app will use visualizations bundled out of the box with READ as well as new kinds of visualizations which we will design in the playground. The app will pull data from real-time REST APIs.

Follow these steps to create your READ app. We recommend opening this tutorial in a separate browser tab while working on your READ app.

Create App

Navigate to App . Create Movies app. Uncheck private button to make this app public.

Create Data Panel & Data Set

Navigate to Data Panel . Create About Movies data panel. Switch to the Data Sets tab and create Movie Ratings dataset. Copy the moving ratings JSON array from this gist and paste into the data section of Movie Ratings.

Create Dashboard

Navigate to Dashboard . Create About Movies dashboard. Switch to the Visualizations tab and create Movie Ratings visualization. In the Data tab, switch the Chart Type to Multibar Horizontal. Populate the basic options for this chart from this gist. Populate the advanced options for this chart from this gist. View your chart in the Visualization tab. Resize your chart by switching to the layout view.

Share Dashboard

Navigate to App . Share your dashboard with the world using the public link for your app.

Key Concepts

A READ App is a collection of data panels and dashboards, which are collections of datasets and visualizations respectively. Set up each visualization in a dashboard using the appropriate data set, chart type, basic and advanced options. Use gists to collaborate with other READ developers. Use the dashboard layout view to resize and drag visualizations. Use the public link for the app to share READ dashboards with the world.
