Time Series Charts

In this section, we will use a variety of chart types available in READ to visualize time-series data.

Create Multibar

Switch to the Visualizations tab. Create visualization and rename it to Sine and Cos Bars. Switch the chart type to Multibar.

Create 5 more Visualizations

Create Sine and Cos Area, SC Line Focus, SC Bars Focus, Sine and Cos Spark, and SC Multi visualizations using the Stacked Area, Line with Focus, Line Bar with Focus, Spark Line, and Multi chart types respectively.

Layout

Switch to the Dashboards tab and layout your dashboard by resizing and repositioning the visualizations.

You can toggle the sidebar using on top of the sidebar.
