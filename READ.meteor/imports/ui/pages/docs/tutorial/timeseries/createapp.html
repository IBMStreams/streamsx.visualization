READ App and Dashboard

In this chapter, we will create and share a READ app with a responsive dashboard.

Opening this tutorial in a separate browser tab while you work on your READ app is recommended.

Create App

Navigate to Dashboard . Switch to the Apps tab. Create App . Double-click on the name of the app and rename it to Sample. Uncheck private button to make this app public.

Create Dashboard

Switch to the Dashboards tab. Create Dashboard and rename it as Time Series.

Create Dataset

Switch to the Datasets tab. Create dataset and rename it to Sine and Cos. Switch the dataset type to URL and set the URL to https://gist.githubusercontent.com/sriumcp/5ca15b9e9b8c51db99cbbadc12047831/raw/b071553834d3c48c0bdde7cced6b2e126040e627/sinecos

Create Visualization

Switch to the Visualizations tab. Create visualization and rename it to Sine and Cos Lines. Switch to the Dashboards tab and resize the visualization appropriately by expanding with bottom right corner. Navigate to Apps tab. You are now ready to share your app and dashboard using the public link for your app.

Key Concepts

  • A READ App is a collection of dashboards. A dashboard is a collection of datasets and visualizations.
  • Make an app public to create a publicly sharable link for the app.
