Create your READ App

In this chapter, we will create and share a READ app with a responsive dashboard.

Opening this tutorial in a separate browser tab while you work on your READ app is recommended.

Create App

Navigate to App . Create Movies app. Uncheck private button to make this app public.

Create Data Panel & Data Set

Navigate to Data Panel . Create About Movies data panel. Switch to the Data Sets tab and create Movie Ratings dataset. Copy the moving ratings JSON array from this gist and paste into the data section of Movie Ratings.

Create Dashboard

Navigate to Dashboard . Create About Movies dashboard. Switch to the Visualizations tab and create Movie Ratings visualization. In the Data tab, switch the Chart Type to Multibar Horizontal. Populate the basic options for this chart from this gist. Populate the advanced options for this chart from this gist. Resize your chart by switching to the layout view.

Share Dashboard

Navigate to App . Share your dashboard with the world using the public link for your app.

Key Concepts

  • A READ App is a collection of data panels and dashboards, which are collections of datasets and visualizations respectively.
  • Set up visualizations using the appropriate data set, chart type, basic and advanced options. See usage info for details on configuring the visualization.
  • Use gists to collaborate with other READ developers.
  • Use the layout view to resize and drag visualizations within a dashboard.
  • Use the public link for the app to share READ dashboards with the world.
