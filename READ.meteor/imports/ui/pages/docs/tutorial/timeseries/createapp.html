READ App and Dashboard

In this section, we will create a READ app with a responsive dashboard

Click on the thumbnail images to see snapshots of READ app at various steps of the tutorial
Click on on top of the sidebar to toggle the sidebar
Open this tutorial in a separate browser tab while you work on your READ app

Create App

Navigate to Dashboard . Switch to the Apps tab. Create App . Double-click on the name of the app, enter Sample as its new name, and click on the Rename button. Uncheck the private button to make this app public.

Create Dashboard

Switch to the Dashboards tab. Create Dashboard and rename it as Time Series.

Create Dataset

Switch to the Datasets tab. Create dataset and rename it to Sine and Cos. From the dropdown list Type, select the dataset type URL. Set the URL to https://gist.githubusercontent.com/sriumcp/5ca15b9e9b8c51db99cbbadc12047831/raw/b071553834d3c48c0bdde7cced6b2e126040e627/sinecos

Create Visualization

Switch to the Visualizations tab. Create visualization and rename it to Sine and Cos Lines. Switch to the Dashboards tab and resize the visualization appropriately by expanding with bottom right corner. You can also view your app and dashboard using the public link for your app available in the Apps tab of your app. To share your READ dashboards publicly, refer to this help topic.
