Cars App

In this chapter, we will work with a variety of chart types in READ such as scatter plot, networks, parallel coordinates, and box plot. We will also create a variety of transforms for cleaning and formatting data.

Create App, Dashboard, Dataset

Create Cars dashboard. Create Cars dataset within the dashboard of type URL with its url set to https://gist.githubusercontent.com/sriumcp/ebe66fbd4bb1f5b16feb0a9e7543f1b5/raw/5fda9a1537b75b1c059b9ce89ee309858dcba737/cars

Create Scatter Plot

Switch to the Visualizations tab. Create visualization and rename it to Sine and Cos Lines. Switch to the Dashboards tab and resize the visualization appropriately by expanding with bottom right corner. Navigate to Apps tab. You are now ready to share your app and dashboard using the public link for your app.

Key Concepts

  • A READ App is a collection of dashboards. A dashboard is a collection of datasets and visualizations.
  • Make an app public to create a publicly sharable link for the app.
