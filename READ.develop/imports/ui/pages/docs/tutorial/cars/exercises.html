Exercises

In this section, you will use the knowledge gained from previous sections to create datasets, visualizations, and dashboards of your choice from public APIs.

The Movie DB

Register at The Movie Database. Request an API key for educational purposes. Create a variety of datasets using TMDB's public APIs. An example of a URL that you can use for dataset creation is https://api.themoviedb.org/3/movie/popular?api_key=your_tmdb_api_key&language=en-US, which returns some currently popular movies. Create a variety of visualizations using these datasets.

BlueMix

Register at BlueMix for a free account. Create the Insights for Twitter service. Use this service to analyze tweets and sentiments, and aggregate sentiments about one of the popular movies from TMDB. Visualize the distribution of these sentiments.
